## 关 0 · 卡塔尔（已完成）

**引擎**：Phaser 3，场景状态机 Boot → Intro → Play → Result  
**文件**：`qatar/game.js` + `qatar/levels.js` + `qatar/style.css`  
**玩法**：
- 角色在地图上行走（虚拟摇杆 + 键盘方向键）
- 捡拾 8 件商品（最多装 6 件）
- 管理水分（💧），水用完会渴死
- 渴死可复活（信使铺写密信 → POST /api/game/secret → +1💧）
- 最终坐船远航到伊朗 → 通关领奖

**通关条件**：到达港口 + 有足够商品  
**奖励档位**：PERFECT ¥20.20 / NORMAL ¥13.14 / HARD ¥6.66 / DEAD ¥0  
**接口**：
- `/api/game/reward/claim` — 通关领奖
- `/api/game/secret` — 渴死复活
- `/api/game/session` — session 兜底

## 关 1 · 伊朗（当前实现 · 需重做）

**引擎**：纯 DOM（无 Phaser/Pixi），完全独立于 level-engine.js  
**文件**：`iran/game.js` + `iran/style.css`  
**现状问题**：❌ 太简单，没有游戏感。纯卡片点击交互，不像卡塔尔关那样有角色在地图上走。

### 当前机制（保留核心逻辑，重做表现形式）

**商品**：8 件（从卡塔尔行李继承），归家之心 ❤️ 不可交易  
**商贩**：5 个波斯商贩，各接受特定商品组合，换得 🐪 骆驼或 💧 水  
**通关条件**：攒够 ≥3🐪 + ≥3💧 → 显示「启程出发」按钮  
**信使铺**：写密信寄回家 → +1💧 奖励  
**通关动画**：沙漠 + 骆驼商队走过 → 通关卡片 + 领奖

### 用户期望的重做方向（2026-07-08）

> "太简单了，没有游戏的感觉。得像卡塔尔那关一样，有一个人在地图上走，然后做商品交换。"

**核心需求**：
1. **地图行走**：像卡塔尔关一样，角色在伊朗沙漠地图上移动（不是卡片列表）
2. **商品交换**：保留 5 商贩 + 8 商品机制，但商贩散布在地图上，走过去才能交易
3. **水分系统**：保留 💧 概念，行走消耗水
4. **骆驼坐骑**：骑上骆驼后每步走 2 格（步行 1 格），更快但需要先用商品换到骆驼

**角色**：复用卡塔尔关已选定的角色（从 localStorage 读取）

### 地点功能

| 地点 | 功能 | 说明 |
|------|------|------|
| **💱 兑换中心** | 换货币 | 把从伊朗带来的商品换成土耳其里拉 ₺，归家之心 ❤️ 不可兑换 |
| **🏪 交易中心** | 卖热气球材料 | 5个商铺：布料、竹条、吊篮、五金、工具 |
| **🎈 热气球组装场** | 组装+起飞 | 检查所有材料 → 组装流程 → 起飞动画 |

### 交互流程

1. **玩家先到兑换中心**：把行李箱商品换成里拉 ₺
2. **去交易中心**：用里拉 ₺ 买热气球材料
3. **买齐所有材料**：布料、竹条×3、吊篮、电线、打火机+燃料、缝纫工具、剪刀
4. **走到组装场**：触发组装流程（复用当前8步逻辑）
5. **组装完成**：点击"出发" → 热气球起飞动画 → 跳转哈萨克斯坦

### 通关条件

- 买齐所有热气球材料
- 完成组装流程
- 点击"出发"触发飞行动画

### 用户期望的增强（2026-07-10）

> "热气球需要额外步骤，比如组装。用于体验真实完整的热气球制作组装过程"  
> "需要玩家点出发才行"  
> "热气球是一段动画"

**核心需求**：
1. **兑换中心**：换货币（把行李箱商品换成土耳其里拉 ₺）
2. **交易中心**：卖东西的商铺集合（卖热气球材料）
3. **热气球组装场**：材料齐全后可进入，组装流程，组装完成后点击"出发"，触发起飞动画

**地图装饰**：岩石烟囱地貌（卡帕多奇亚特色）

## 共享状态（localStorage）

| Key | 用途 |
|-----|------|
| `silkroad_session_id` | 游戏 session（领奖/密信用） |
| `silkroad_nickname` | 玩家昵称 |
| `silkroad_luggage` | 卡塔尔行李（商品 id 数组），传给伊朗关 |
| `silkroad_cleared_levels` | 已通关关卡 id 数组 |
| `silkroad_secret_sent_{level}` | 该关是否已发密信 |
| `silkroad_claimed_{sid}_{level}` | 该关是否已领奖 |

## 技术栈

- **Phaser 3.80**（卡塔尔关）
- **Pixi.js**（关 2-4 旧引擎，可能需替换）
- **纯 DOM**（伊朗关当前实现）
- **Flask 后端**：session / reward / secret API
- **BGM**：全局 `<audio id="silk-road-bgm">`，各关控制播放
