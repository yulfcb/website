# 哈萨克斯坦·草原套马 - 设计文档

## 核心流程

**套马 → 骑马买补给 → 出发去新疆**

---

## 场景 1：套马（TamingScene）

### 目标
在草原上套中 1 匹野马

### 玩法
- 草原上有 5-8 匹野马随机奔跑
- 玩家手持套马索（绳子 + 圆圈）
- 操作方式：
  - 点击/拖拽甩出套马索
  - 套中马匹（圆圈套住马脖子）
  - 需要连续套中 3 次才能驯服

### 难度递增
- **第 1 次**：马匹速度慢
- **第 2 次**：马匹加速躲避
- **第 3 次**：马匹快速移动 + 变向

### 失败机制
- 套空 5 次 = 失败，显示"再试一次"按钮
- 或者限时 30 秒，超时失败

### 视觉设计
- 广阔草原背景（绿色渐变）
- 远处雪山轮廓
- 野马群奔跑动画（Phaser sprite/tween）
- 套马索动画（抛物线 + 圆圈收缩）

### 通关条件
成功驯服 1 匹马

---

## 场景 2：骑马探索 + 买补给（PlayScene）

### 目标
走访 5-6 个蒙古包，购买新疆之旅的补给

### 地图设计
- 1280×720 草原地图
- 玩家骑马移动（速度比步行快 2 倍）
- 6 个蒙古包驿站分布在不同位置

### 蒙古包驿站

| 驿站 | 商品 | 价格 | 用途 |
|------|------|------|------|
| 🏇 马具驿站 | 升级马鞍 | 100₺ | 速度 +30% |
| 🏹 弓箭驿站 | 弓箭 | 80₺ | 路上打猎用 |
| 🥛 奶制品驿站 | 马奶酒 | 60₺ | 补充体力 +20 |
| 🦅 鹰猎驿站 | 猎鹰 | 120₺ | 增加视野范围 |
| 🧥 毛皮驿站 | 保暖衣物 | 150₺ | 新疆雪山防寒 |
| 🗺️ 地图驿站 | 草原地图 | 50₺ | 显示最优路线 |

### 货币系统
- 从土耳其关卡继承里拉 ₺
- 可以在"兑换中心"把里拉换成哈萨克坚戈 ₸（汇率 1:5）

### 补给需求
- **必须买齐**：保暖衣物 + 马奶酒（否则无法进入新疆雪山）
- **可选**：弓箭、猎鹰、马鞍升级（增加分数）

### 通关条件
买齐必需补给 + 走到地图出口（左上角"→ 新疆"）

---

## 场景 3：出发去新疆

### 触发条件
- 保暖衣物 ✓
- 马奶酒 ✓
- 走到出口位置

### 动画
- 骑马冲向远方
- 背景从草原渐变到雪山
- 显示"🏔️ 新疆·天山"标题
- 3 秒后跳转 `/games/silk-road/level/4`

---

## 技术实现

### 场景结构
```javascript
BootScene      // 加载资源
TamingScene    // 套马场景
PlayScene      // 骑马探索 + 买补给
// (无独立出发场景，出发动画在 PlayScene 末尾)
```

### 复用组件
- `common.js` 的角色绘制（`buildAvatarSprite`）
- 货币系统（从土耳其继承）
- 行李系统（`localStorage.silkroad_luggage`）
- HUD 布局（水分/余额/行李/材料）

### 新增组件
- 套马索动画（Phaser Graphics 绘制抛物线 + 圆圈）
- 野马 AI（随机移动 + 躲避逻辑）
- 蒙古包驿站（类似土耳其的商铺系统）

## 与现有关卡的集成

### 数据流
1. **输入**（从土耳其继承）：
   - `localStorage.silkroad_luggage`：行李物品
   - `localStorage.silkroad_coins`：里拉余额

2. **输出**（传递给新疆）：
   - `localStorage.silkroad_luggage`：更新后的行李
   - `localStorage.silkroad_kazakhstan_items`：哈萨克斯坦购买的物品
   - `localStorage.silkroad_cleared_levels`：追加 3

### 路由
- 入口：`/games/silk-road/level/3`
- 出口：`/games/silk-road/level/4`

## 美术风格

### 色彩方案
- 背景：草原绿渐变（#7CB342 → #558B2F）
- 天空：淡蓝（#81D4FA）
- 雪山：白色（#FFFFFF）+ 灰色阴影（#B0BEC5）
- 蒙古包：米白（#F5F5DC）+ 棕色装饰（#8D6E63）

### 动画要求
- 野马奔跑：4 帧循环
- 套马索：抛物线 + 圆圈收缩
- 骑马移动：2 倍速 + 尘土粒子效果
- 蒙古包：静态 + 烟雾粒子
