### 1.3 兑换中心（Step 0）— 换货币

**功能**：把从伊朗带来的卡塔尔/伊朗商品换成土耳其里拉 ₺

- 从 `localStorage['silkroad_luggage']` 读取伊朗带来的卡塔尔礼物
- 每件物品有对应的里拉兑换价（参考 EXCHANGE_RATES）

## 3. 设计要求（用户明确要求）

### 3.1 热气球体验要真实完整

- **不是简单的"交易完成"抽象结果**，而是完整的"得到热气球 → 坐热气球 → 到达哈萨克斯坦"体验
- 热气球需要**额外步骤**（组装），不是直接获得
- 玩家需要体验**真实完整的热气球制作组装过程**
- 最后需要玩家**主动点击"出发"按钮**才能起飞
- 起飞是**一段动画**（上升 + 飘动 + 字幕）

### 3.2 货币系统

- 进入土耳其后，先去**兑换中心**用伊朗带来的物品换土耳其里拉
- 用里拉在市场**购买材料**（不是以物易物，不是直接从背包选）
- 购买完成后才进入制作流程

### 3.3 流程设计原则

- **不是"完成3个任务 → 解锁下一关"**的抽象逻辑
- 而是**有仪式感、有体验感的完整旅程**：
  1. 换钱（经济准备）
  2. 买材料（物质准备）
  3. 制作组装（手工过程）
  4. 点火测试（安全检查）
  5. 点击出发（主动决定）
  6. 起飞动画（视觉享受）

## 1. 剧情梗概（约 480 字）

**主线：东方商人 × 西方终点**

你——一名穿越了卡塔尔沙海、走过伊朗沙漠的丝路商人，终于站在 1453 年奥斯曼帝国治下的**伊斯坦布尔**面前。博斯普鲁斯海峡的风带来咸湿的水汽，金角湾里桅杆如林；穿过加拉塔大桥，便是大巴扎的铜门——**1461 年起便是丝绸之路西端最大集散地**。

你此行的最后一笔生意：把从中国/波斯带来的**丝绸、瓷器、香料**换成欧洲人的**威尼斯玻璃、佛罗伦萨绒布、拜占庭金币**。但大巴扎不是桃源——

- 街口站着的**希腊语翻译官**收 5 阿克切（akçe，白银币）才肯替你开口
- 集市上**威尼斯商人 Marco** 和**热那亚商人 Giovanni** 都在跟你抢同一批货，价格随时波动
- 苏丹的**海关官员（Kapıcı）** 查得严——**亵渎宗教禁忌**会被当众羞辱并扣 20 声望
- **伊斯兰教徒 vs 基督徒**的交易区是分开的，走错区会被赶走

完成 **3 笔跨文化交易**、**净赚 ≥ 50%**（本金 100 阿克切 → 净利 ≥ 50），再找到隐藏在圣索菲亚大教堂穹顶下的**拜占庭金币**（隐藏宝藏），便可登船西渡——往威尼斯、往君士坦丁堡以西、往你口中的"故乡"。

**失败线**：声望崩塌、被赶出集市、破产、或被海关官员关铺。

## 2. 游戏机制（仿 qatar/iran 三段式）

### 2.1 总体节奏

```
INTRO  ─►  PLAY（行走+交易+换币） ─►  RESULT  ─►  威尼斯动画 → /games/silk-road/world-map
   │                                  │
   │                                  ├─ PERFECT ¥20.20  全部交易+隐藏宝藏
   │                                  ├─ NORMAL  ¥13.14  完成 3 笔 + 净利 ≥ 50%
   │                                  └─ HARD    ¥6.66   完成 1 笔 + 净利 ≥ 20%
   └─ 不限时，无 can_fail（不会渴死，纯交易游戏）
```

### 2.2 三层玩法

#### A. **核心玩法：跨文化交易**（必玩）

- 玩家带来：从 `localStorage['silkroad_luggage']` 读关 1 留下的商品（id 0-7，⚠️ 归家之心 ❤️ 仍不可交易）
- 玩家带走的：威尼斯玻璃 🪟 / 佛罗伦萨绒布 🧵 / 拜占庭金币 🪙 / 香料礼盒 🎁（西方商品入 luggage）
- 机制：3 轮讨价还价（受 qatar 礼物 modal 启发）
  - **第 1 轮**：开价（玩家出 1 件东方货 + 阿克切）
  - **第 2 轮**：NPC 还价（涨跌 ±30%）
  - **第 3 轮**：成交 / 拒绝 / 加换（+1 件东方货换更多西方货）

#### B. **副玩法：货币兑换**（推荐玩）

- 在**金角湾兑换所**用东方货物换 **akçe**（白银币 / akçe）
- 汇率波动：每 30 秒 ±5%，玩家低买高卖可套利
- 风险：5% 概率遇到**假币**，必须用"咬币测试"小游戏识别

#### C. **探索玩法：大巴扎迷宫**（可选）

- 网格地图（参考 iran 的 place chip）
- 5 个分区：香料区、地毯区、珠宝区、书商区、清真寺
- 隐藏任务：圣索菲亚大教堂穹顶下的拜占庭金币

### 2.3 状态系统

| 指标 | 初始值 | 最大值 | 失败阈值 |
|------|--------|--------|----------|
| 💰 阿克切（akçe） | 100 | ∞ | < 10 |
| ⭐ 声望（reputation） | 70 | 100 | < 20 |
| 🕐 时间（剩余回合） | 30 回合 | — | 0 = 收市 |
| 🎒 东方货（easternCargo） | 从 luggage 读 | 8 件 | 卖完 |
| 🛍️ 西方货（westernCargo） | 0 | 5 件 | — |

### 2.4 行动点机制（取代回合制）

- 玩家每做一件事消耗 1 个行动点（AP）：移动 1 格 / 开价 1 次 / 兑换 1 次
- 30 AP = "一天的开市时间"，每个 AP = 现实 6 秒（玩家不会被催得太紧）
- AP < 5 时顶部 HUD 变红："⚠️ 集市快收摊了！"

```
NORMAL  通关要求: 净利率 ≥ 50%  AND  完成 3 笔跨文化交易

```

## 10. 关键设计权衡

1. **不引入新的失败机制（渴死/超时 can_fail）**：本关是贸易模拟，时间用 AP 而非真实秒数；保留纯交易乐趣
2. **文化敏感度是软机制**：3 次才死，比"一次就死"更宽容；新手玩家可犯错
3. **竞争 NPC 制造紧迫感**：Marco 每 10 AP 抢货，逼玩家"快买快卖"
4. **翻译官是付费 NPC**：考验玩家"是否值得花 5 akçe 节省 1 轮沟通失败"
5. **隐藏金币驱动探索**：玩家为 PERFECT 主动走遍地图 → 提升重玩价值
